# Навигация по документам

Короткая навигация. Читать только релевантные файлы.

## Основные
- `agent_docs/architecture.md` — архитектура парсера: компоненты (коллекторы, нормализатор, обогатитель, экспортёры), потоки данных, технологии.
- `agent_docs/pipeline.md` — идеальная последовательность создания клиентских списков: от источников до экспорта в Bitrix24 и AI Sales Factory. Форматы данных, маппинг полей, фазы запуска.
- `agent_docs/data-sources.md` — реестр конкретных источников, URL и ссылок для парсинга. Сюда добавлять новые источники данных.
- `agent_docs/adr.md` — архитектурные решения и компромиссы; использовать при важных решениях.
- `agent_docs/development-history.md` — журнал итераций; смотреть последнюю запись. Архив: `agent_docs/development-history-archive.md`.

## Правила и гайды
- `agent_docs/guides/dod.md` — критерии завершенности (DoD).
- `agent_docs/guides/environment-setup.md` — настройка окружения; применять при инициализации проекта.
- `agent_docs/guides/logging.md` — логирование скриптов/интеграций.
- `agent_docs/guides/archiving-and-temp.md` — архивация и временные файлы.

## Контекст продукта
- Репозиторий `pass24-knowlege-base` — база знаний: продукт, ICP, конкуренты, методология продаж, HADI-циклы.
- Репозиторий `pass24-ai-sales` — AI Sales Factory: 12 агентов, Lead модель, CHAMP-скоринг, Bitrix24 интеграция.
